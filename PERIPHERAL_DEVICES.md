## Joystick

See `JoystickController.h`:

* `GPIO33` - `X`
* `GPIO32` - `Y`

## Buttons

See `ButtonsManager.h`:

* `GPIO35` - `Menu Button`

## Screen

See `menuconfig -> Component config -> LVGL TFT Display controller -> Display Pin Assignments`:

* `GPIO13` - `MOSI`
* `GPIO12` - `MISO`
* `GPIO14` - `CLK`
* `GPIO17` - `DC`
* `GPIO25` - `Reset`
* `GPIO26` - `Backlight`
